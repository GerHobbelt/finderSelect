jQuery finderSelect

Provides the ability to add file manager (Finder) type highlighting to all elements based on the standard single click, command+click/ctrl+click, command+drag/ctrl+drag and shift+click methods. This plugin will by default add a `selected` class to all clicked elements.

View on Github

Tested on IE6+, Firefox, Safari & Chrome. Single click only on touch devices.

Advanced Usage

This advanced demo shows how finderSelect can be used as a component to create a file explorer. Make sure to try Ctrl/Cmd+Click, Shift+Click, Ctrl/Cmd+Click&Drag and Right Click on the table after you have made a selection.

$(function() {
    $('#demo5').finderSelect({children:'tr'});

    $("#demo5 tr").dblclick(function() {
        alert( "Double Click detected. Useful for linking to detail page." ); // Enable double click
    });

    $.fn.finderSelect.highlight = function(el, cssClass) {
        el.addClass(cssClass);
        el.find('input').prop('checked', true); // Add functionality to check checkboxes inside selected rows

    };

    $.fn.finderSelect.unHighlight = function(el,cssClass) {
        el.removeClass(cssClass);
        el.find('input').prop('checked', false); // Add functionality to uncheck checkboxes inside unselected rows
    };

    $("#demo5 tr input").click(function(e) {
        e.preventDefault();
    });

});
