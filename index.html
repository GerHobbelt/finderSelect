jQuery finderSelect

Adds the ability to use file manager (Finder) type highlighting to all elements based on the standard single click, command+click/ctrl+click, command+drag/ctrl+drag and shift+click methods. This plugin will by default add a `selected` class to all clicked elements.

$(function() {
    $('#demo1').finderSelect({children:'tr'});
    $("#demo1").on("finderSelectUpdate", function(event){
        $('#demo1-count').html($(this).find('tr.selected').length)
    });
    $("#demo1-trash").click(function(){
        $('#demo1').find('tr.selected').remove();
        $('#demo1').trigger('finderSelectUpdate');
    });
});
